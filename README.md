# Retail Sales ETL & Analytics Project
# Overview
This project demonstrates an end-to-end data engineering and analytics workflow, including:

• Data extraction from multiple source formats (Excel, CSV, TXT)
• ETL pipeline development using SSIS
• Slowly Changing Dimension (Type 2) implementation
• Data warehouse schema design
• Error handling and logging mechanisms
• Interactive Power BI dashboard creation

# Technologies Used

• SQL Server
• SSIS (ETL Development)
• Power BI
• Git Version Control

# ETL Features Implemented

✔ Staging → Dimension → Fact architecture
✔ Lookup transformations for key resolution
✔ SCD Type 2 for Customer Dimension
✔ Incremental load logic
✔ Duplicate prevention logic
✔ Derived columns for audit tracking
✔ Error row redirection to flat files

# Power BI Dashboard Features

✔ KPI cards for Sales Metrics
✔ Top-N dynamic filtering
✔ Drill-through navigation
✔ Tooltips with detailed insights
✔ Hierarchical date analysis
✔ Interactive slicers (City, Category, Year)

# Business Insights Provided

• Top selling products
• Sales trends over time
• Regional performance analysis
• Customer purchasing patterns
